# 目录树生成器 (Tree Generator)

[English](./README.md) | [简体中文](./README.zh-CN.md)

---

**Tree Generator** 是一款轻量级的 VS Code 生产力插件。它能够为复杂的工程项目快速生成结构清晰的目录树文件，
集成了 `.gitignore` 规则解析、资源管理器右键菜单，并为 `.tree` 专属文件提供了原生的语法高亮与折叠支持。

## 主要功能

- **上下文感知生成：** 在侧边栏资源管理器中右键点击任意目录，即可精准生成该路径下的局部 `tree.tree` 文件。
- **智能 `.gitignore` 解析：** 自动向上追溯并定位工作区内的 `.gitignore` 文件，精准应用相对路径与全局过滤规则。
- **重型目录收敛：** 针对 `node_modules`、`.git` 等重型依赖文件夹，自动收敛并隐藏内部细节，保持树状图的高可读性。
- **`.tree` 专属语言支持：**
  - **语法着色：** 为文件夹、文件、树形连接符以及注释提供语义化高亮。
  - **代码折叠：** 基于缩进层级，提供原生的节点折叠与展开功能。
  - **快速注释：** 支持使用 `Ctrl+/`（或 `Cmd+/`）快速切换行注释 (`#`)。

## 使用指南

### 方式一：右键菜单（推荐）
1. 打开 VS Code 左侧的资源管理器。
2. 右键点击目标文件夹。
3. 在弹出的菜单中选择 **"Generate Tree"**。

### 方式二：命令面板
1. 呼出命令面板 (`Ctrl+Shift+P` / `Cmd+Shift+P`)。
2. 输入并执行 **"Generate Tree File"**。此操作将默认以当前工作区根目录为基准生成。

## 默认过滤规则

在未检测到 `.gitignore` 文件的情况下，插件内置了以下安全过滤名单以避免性能问题：
`['.git', '.venv', 'node_modules', '__pycache__', 'dist', 'build']`
